# Brain Dead Project

## 프로젝트 소개

- 제가 좋아하는 브랜드인 'Brain Dead' 홈페이지를 클론 코딩하며 기본적인 CRUD를 구현하였습니다.
- API를 사용하여 구현한 기능으로는 로그인, 회원가입, 상품목록, 상세페이지, 장바구니, 프로필 조회 및 수정, 포인트 충전, 구매입니다.
- 팀은 프론트엔드 1명, 백엔드 1명으로 구성되어있습니다.
